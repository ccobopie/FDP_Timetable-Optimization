How to Use TimetableApp

1. Create Tasks: Add your tasks with deadlines and preferences from the "Create Task" page.

2. Plan Weekly: Use the "Timetable" page to visualize your week with automatically scheduled tasks.

3. Import from Canvas: Go to Canvas config and import tasks from your selected courses.

4. Edit or Delete: Modify tasks anytime from the "Tasks" page.

5. Conflicts: If there's overlap, the system will notify you and suggest adjustments.
